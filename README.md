# عبدالخالق-فيزياء (physics-diagnostic-test)

مشروع تعليمي لبوابة الفيزياء التشخيصية ودورات إثرائية وتمارين للطلاب والمعلمين.

## شرح عام

يحتوي المشروع على:
- صفحة رئيسية بوابة الدخول.
- ملفات style.css و script.js للتنسيقات والوظائف العامة.
- مجلدات (assets) للصور والملفات والخطوط.
- قسم اختبارات (exams) متنوعة.
- قسم قاموس فيزيائي (dictionary).
- قسم الدروس (lessons).
- أسئلة تدريب (questions) حسب المعايير.
- لوحة تحكم الطالب (student-dashboard).

جميع الملفات حالياً هيكلية وجاهزة لتعبئة المحتوى لتسريع التطوير.
